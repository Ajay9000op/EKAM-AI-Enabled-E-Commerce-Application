# data/

CSV datasets are stored here (Phase 2).
Later phases may add SQLite and migration scripts.

## Datasets
The synthetic generator in this repo outputs:
- `products.csv`
- `users.csv`
- `interactions.csv`

All files are generated by `data/generate_datasets.py`.

## Regenerating datasets
1) From the project root, run:

```bash
python data/generate_datasets.py
```

2) (Optional) Set a seed for deterministic output:

```bash
python data/generate_datasets.py --seed 42
```

3) (Optional) Validate that the generated CSVs have the expected schema and constraints:

```bash
python data/generate_datasets.py --validate
```

4) (Optional) Tune generation parameters:

```bash
python data/generate_datasets.py \
  --num_users 500 \
  --num_products 300 \
  --num_categories 20 \
  --days_back 180 \
  --min_interactions_per_user 10 \
  --max_interactions_per_user 80
```

## Output locations
Generated CSVs are written directly into this folder:
- `data/products.csv`
- `data/users.csv`
- `data/interactions.csv`
